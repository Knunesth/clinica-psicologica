# Mente Serena — Psicologia Clínica

Landing page institucional para a clínica fictícia de psicologia **Mente Serena**.

## Tecnologias

- React 19
- Vite
- CSS puro (sem frameworks)
- Google Fonts (Inter + Playfair Display)

## Seções

- **Hero** — Apresentação da clínica com CTA
- **Sobre** — Missão e valores da clínica
- **Serviços** — Terapia Individual, de Casal, Infantil, Avaliação Psicológica, Orientação Profissional, Psicoterapia Online
- **Equipe** — Psicólogos fictícios com especialidades
- **Depoimentos** — Relatos fictícios de pacientes
- **Contato** — Formulário, telefone, e-mail, WhatsApp

## Como rodar

```bash
npm install
npm run dev
```

## Deploy

O projeto é publicado via GitHub Pages usando o branch `gh-pages`.

```bash
npm run build
npx gh-pages -d dist
```
